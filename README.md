# Target Darts Platform
Play x01, Cricket & Chaos modes with NPC, Camera Scoring, Manual Input, Animations, Sound Effects, and Statistics.

## Features
- x01 Games: customizable start score, sets, legs, MPR & averages.
- Cricket: track hits per number, sets, legs.
- Chaos Mode: random zones, effects, animations.
- Camera auto scoring + manual fallback.
- NPC bot opponents.
- Full statistics: darts thrown, average, MPR, hits.
- Multi-player support 2-8 players.

## How to Run
1. Clone repo
2. Open `index.html` in a modern browser
3. Use camera for automatic scoring or manual input
